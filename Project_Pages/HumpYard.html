Hump Yard

The Hump Yard game is a board game about railroad classification yards. Players compete to win train sorting contract by being able to sort trains in fewer moves than there opponents. This board game is currently in development and has received funding through the Henry Armero Award. For more information about Hump Yards, please see Wikipedia Page. For more information about the Henry Armero Award, please see: Award Website. The following image shows the initial prototype of the Hump Yard board game.
I am currently designing rules for the game along with doing play testing. Currently the illustrations are being produced and the game rules are being refined. The first full version of the game will be displayed to the public in mid October.
